1
基础语法
变量、数据类型、if/for/while、函数定义、列表/字典操作
3~5 天
2
文件与模块

2~3 天
3
实用库速学
json（数据格式）、requests（HTTP 请求）、os/pathlib（路径操作）
requests json 3~4 天

调试能力

贯穿始终
✦

写一个脚本：读取本地 CSV → 筛选数据 → 输出结果到新文件
动手项目
PHASE 02 数据处理能力 ⏱ 2~3 周

AI 工作中 80% 的时间在处理数据，这一步决定你的工作效率。

1
Pandas 核心操作
DataFrame 创建、读取 CSV/Excel、筛选过滤（loc/iloc）、分组聚合（groupby）、缺失值处理
Pandas 7~10 天
2
SQL 基础查询
SELECT / WHERE / JOIN / GROUP BY，能对数据库做基本查询即可
SQL 3~5 天
3
JSON 数据处理
解析嵌套 JSON、API 返回值提取、JSON ↔ DataFrame 互转（AI 接口全是 JSON）
2~3 天
✦
🏋️ 练手项目
用 Pandas 清洗一份真实数据集（如 Kaggle 数据），生成统计摘要
动手项目
🏁
基础能力达标 — 可以开始接触 AI 实战了
到这里大约 1~1.5 个月，你已经能写脚本、处理数据

3~4 天

动手项目

企业最刚需的场景 —— 让大模型读懂你的私有数据。会 RAG 就能解决 70% 的企业需求。

2~3 天
2

3
文档处理管线
PDF/Word/网页的文本提取、文档切分（Chunking 策略）、元数据管理

用户提问 → 向量检索 → 拼装 Prompt → 大模型生成回答 → 引用溯源
3~5 天
✦
🏋️ 练手项目
搭建一个"企业知识库问答系统"：上传 PDF → 自动索引 → 智能问答
动手项目简历亮点

到这里大约 3 个月，可以开始投递简历了！

总计约 3~5 个月 · 可胜任 AI 应用开发工程师 / LLM 工程师岗位

💡 给小白的四条建议

① 项目驱动学习：每个阶段都有练手项目，千万别只看教程不动手。做出来的东西就是你的简历和作品集。

② 不要纠结"学完再开始"：Python 学 70% 就去碰 Pandas，Pandas 学 70% 就去调 API。实战中遇到不会的再回头补，效率最高。

③ 善用 AI 辅助学习：遇到报错直接丢给 ChatGPT / Claude 解释，这本身就是在练"和 AI 协作"的能力，也是未来工作日常。

④ 先广后深：先把整条链路跑通（能调 API → 能做 RAG → 能搭界面），再回头深入某个环节。广度优先，T 型发展。
